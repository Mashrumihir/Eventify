# 🎉 Eventify - Event Management System (React)

## 📌 Problem Statement  
Build a system to manage event planning, booking, and participation from creation to completion.

---

## 🎯 Objective  
To develop a centralized platform that simplifies event organization, improves booking management, and enhances user engagement using a modern frontend framework.

---

## ✅ Expected Outcome  
A complete event lifecycle system with:  
- Event creation & management  
- Booking & tracking system  
- Role-based dashboards  
- Analytics & reporting  

---

## ⚠️ Constraints & Considerations  
- Must support multiple events simultaneously  
- Role-based access (Admin / Organizer / User)  
- Real-time booking updates (via API)  
- Consistent and reliable event data  

---

## 📅 Progressive Milestones  
- **T+4:** UI setup + event listing (React components)  
- **T+8:** Booking system + organizer dashboard  
- **T+16:** Search/filter + analytics integration  
- **T+24:** Payment + notifications + UI polish  

---

## 📌 Project Overview  
**Eventify (React Version)** is a frontend-driven Event Management System built using **React.js**, designed for fast, dynamic, and responsive user interaction.

It connects with backend APIs to manage events, bookings, and user data efficiently.

---

## 👥 User Roles  

### 👑 Admin  
- Manage users  
- Monitor all events  
- View analytics & reports  
- Control event listings  

### 🎤 Organizer  
- Create & manage events  
- Track bookings  
- View dashboard (sales & visitors charts)  
- Update event details  

### 👤 User  
- Browse events  
- Book tickets  
- View booking history  
- Manage profile  

---
